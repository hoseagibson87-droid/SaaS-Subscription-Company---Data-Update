# 📊 SaaS Customer Churn & Revenue Analysis

## 📌 Overview
This project analyzes customer churn and revenue trends for a SaaS business.

---

## 🎯 Business Problem
The goal is to understand:

- What is driving fluctuations in revenue?
- What factors contribute to customer churn?
- How do region and product impact churn?

---

## 🧩 Project Approach

### 1. Data Quality Assessment
- Identified missing data (up to 73%)
- Determined dataset was not reliable
- Recommended upstream fixes

### 2. Data Cleaning
- Standardized column names
- Converted data types
- Handled missing values

### 3. Exploratory Data Analysis
- Revenue trends over time
- Churn by region
- Customer behavior analysis

---

## 📊 Key Insights

- West region has highest churn (~36%)
- East region has lowest churn (~20%)
- Product differences have minimal impact
- Product mix only partially explains churn

---

## 💡 Recommendations

- Investigate regional differences in the West
- Improve data collection processes
- Monitor churn risk indicators

---
